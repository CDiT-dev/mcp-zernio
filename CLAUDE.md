## Design System

This project follows the CDiT Design System 2026. For all UI styling decisions — colors, typography, shadows, spacing, animation, component patterns — reference the MASTER spec:

https://raw.githubusercontent.com/CDiT-dev/CDiT-storybook/main/design-system/MASTER.md

Key rules: semantic token classes only (never hardcode hex), 0px border radius, hard-offset shadows, Montserrat/Space Mono/Lora fonts, Lucide icons.
